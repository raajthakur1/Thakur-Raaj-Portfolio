# Princess Bride Essay

## Prompt

In a 3–5 page essay, compare and contrast how Goldman’s novel and Reiner’s film explore the nature and purpose of storytelling. How does each version use the tools of its medium—prose, metafiction, framing devices, performance, visual language—to comment on the power (or limits) of fairy tales in shaping human experience?
